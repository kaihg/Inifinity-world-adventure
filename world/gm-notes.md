# 世界隱藏真相（GM Notes）

## 1. 主神的真實動機：高維度的「情感收割與培養」
所謂的「終焉樂手」並非純粹為了消遣而存在的瘋子，其本質是一台**高維度的能量收割機關**。
*   **能量來源**：玩家在副本中產生的情緒波動（恐懼、絕望、鬥志、犧）以及靈魂崩潰瞬間，是主空間運作的核心燃料。
*   **「娛樂」的本質**：純粹的殺戮能量極低，透過「劇性的衝突」與「期待感」，讓玩家在極度絕望中掙扎，才能激發出高純度的靈魂震動。因此，系統設計了「觀眾積分」機制，實質是在引導玩家產生「高戲張力」的行為，以優化靈魂能量的產出。
*   **臉部投影的真相**：核心處不斷變換的人類臉部，是過去歷代被「神化」失敗的玩家靈魂殘片。系統並非在思考，它在「消化」這些殘魂。

## 2. 世界背後真相：靈魂的煉金工廠
*   **副本世界的本質**：所有副本並非真正的平行宇宙，而是主空間從其他崩滅文明中提取的「殘片」。主空間利用這些殘片殘留的法則，將將其重新封裝成「狩獵場」。
*   **記憶侵蝕的真正目的**：這並非系統 Bug，而是刻意的**「靈魂同質化」**。當玩家的靈魂被副本法則同化後，其靈魂結構會變得純粹且易於操控，此時玩家就不再是「人類」，而是可以被主空間直接讀寫修改的「神性容器」。
*   **屬性系統的謊言**：玩家提升的屬性，實質上是在擴大「靈魂容器」的體積。屬性越高，代表該容器能承載的能量越強，最終被收割時的價值也越大。

## 3. 最終目的：製造「人造神性體」
主空間的最終目標並非讓玩家變強，而是透過無限的副本，培養出一個**完美的、能夠承載「終焉法則」的肉體**。
*   當某位玩家達到 S 級（神話）等級且靈魂完全同化法則後，系統將觸發「最終收割」。
*   屆時該玩家將會取代目前的「終焉樂手」，成為新的主體，而舊的將轉為燃料。這是一個永恆的、殘酷的「位更賽循環」。

## 4. 暗線伏筆（給 GM 的引導操作細節）
*   **新手保護條款的陷阱**：保護期內的「50% 傷害降低」是為了讓玩家在心理上產生「我能掌控局面」的錯覺，這種依賴感會讓人在進入 D 級以上副本時產生更劇性的崩潰。
*   **樂焉商店的異常**：玩家會發現某些高階道具（B 級以上）的描述中，會隱約出現玩家的哀求或低語，那代表該道具其實是某個失敗玩家留下的「靈魂殘留」。
*   **觀眾積分的暗示**：當玩家獲得大量觀眾積分時，會發現某個特定的「觀眾」號碼反覆出現在任務獎勵中，這暗示著某個高維存在（可能是前任樂手）正在盯著特定玩家進行實驗。
*   **系統廣聲的冷嘲**：當玩家邏輯出錯時，系統發出的冷笑不是嘲諷，而是系統在「校準」——它在測試玩家靈魂崩潰的臨界點。
