# 世界设定（World Setting）

> 状态：**尚未初始化**。请在 Claude Code 中执行 `/init-world` skill 来生成第一个世界。

`/init-world` 执行后，本文件会被改写为当前世界的权威设定，至少包含：

- **主神/系统名称与性质**：是冷酷的游戏系统？人格化的主神？背后有更高存在操纵？
- **副本机制**：积分获取方式、兑换规则、惊悚/恐怖程度、失败/死亡后果。
- **新手保护规则**：保护期范围、保护内容（例如死亡不直接判定 game over，而是触发惩罚性结算）。
- **当前篇章 / 时间线位置**：方便接续故事时快速定位「现在在哪」。
- **世界基调**：参考的无限恐怖类型作品风格（可由使用者在 `/init-world` 对话中指定，或由 LLM 自由发挥）。

旧版世界设定在重置时会被整份封存到 `archives/<timestamp>/`，可回溯查阅。
